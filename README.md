## # Requirements
* Install `Nodejs` <= v20.2.0

## # How to run ?
* **Start App** `$ npm start`
* **Testing By Typescript** `$ npm test:ts`
* **Build App** `$ npm run build`

## # Importing Allies
* `@/*` root directory is (`./src`)
* `~/*` root directory is (`./public`)
